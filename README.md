</div>

---

### 📌 About The Project
EventHub is an innovative graduation project designed to modernize and simplify event management through a smart digital platform.

The application connects all event participants — including organizers, sponsors, speakers, and attendees — within one integrated mobile system.

**Key capabilities:**
* 🔍 Discover and explore events
* 💳 Book tickets online
* 📍 View event locations using Google Maps
* 🔔 Receive notifications and reminders
* 📁 Access QR-based digital tickets

---

### 🚀 Features
* **👤 Authentication:** Login, Register, OTP Verification, and Persistent Login.
* **🎟️ Event Booking:** Instant booking with QR Code generation.
* **❤️ Favorites:** Save and manage your favorite events.
* **🗺️ Maps Integration:** Interactive Google Maps view for locations.
* **🌍 Localization:** Full support for Arabic and English.

### 📂 Project Structure
```text
lib/
├── core/
│   ├── api/             # API services & Dio config
│   ├── constants/       # Assets & App keys
│   ├── services/        # Location & Storage
│   └── theme/           # App colors & styles
├── features/            # Feature-based modules
│   ├── auth/            # Login, Signup, OTP
│   ├── home/            # Home, Search, Maps
│   ├── ticket/          # Booking & QR
│   └── profile/         # Settings & Localization
├── models/              # Data models
├── providers/           # State management
└── l10n/                # Localization files (AR/EN)
```
